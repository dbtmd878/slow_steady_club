# Slow Steady Club 클론 코딩

제가 좋아하는 쇼핑몰을 클론 코딩해보았으며, 프로젝트를 진행하다 문법과 API사용등 부족한 점이 많이 보여 해당 프로젝트를 중단하였습니다. 

## 구현 내용

날씨 API를 불러와 현재 날씨를 표현해주며 메뉴를 클릭했을 때 나오는 페이지를 순수 JS로만 작성해서 구현해보았다.
